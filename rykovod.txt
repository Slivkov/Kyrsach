Инструкция к применению и возможности приложения.

Использование приложения «Список услуг медицинского центра» очень просто и интуитивно понятно.
При запуске приложения нам показывается таблица с двумя услугами, указанными по умолчанию.
Под услугами располагаются кнопки:
1. «Добавить услугу» - при нажатии появляется окно с полями ввода с информацией о добавляемой услуге.
После ввода следует нажать «Добавить», после чего новая услуга будет добавлена в таблицу. 
Если решение добавлять услугу было изменено, то стоит нажать на «Отмена».

2. «Удалить услугу» - при нажатии удаляет выбранную услугу. 
По умолчанию выбрана самая верхняя услуга. 
Выбрать самостоятельно можно посредством нажатия на услугу левой кнопкой мыши, 
либо используя стрелочки «вверх» и «вниз» на клавиатуре.

3. «Сохранить» - при нажатии открывает проводник, 
где нужно выбрать место сохранения и название файла. 
По умолчанию открывается папка build проекта. 

4. «Загрузить» - при нажатии открывает проводник, 
где нужно выбрать файл для считывания информации. 
По умолчанию открывается папка build проекта.

5. «Копировать» - при нажатии копирует выбранную услугу. 
По умолчанию выбрана самая верхняя услуга. 
Выбрать самостоятельно можно посредством нажатия на услугу левой кнопкой мыши, 
либо используя стрелочки «вверх» и «вниз» на клавиатуре.

6. Выход – при нажатии завершает работу приложения.

Но на кнопках приложение не заканчивается. Также имеются следующие возможности:

1. При двойном клике левой кнопкой мыши на любой из параметров услуги,
 откроется окно редактирования выбранной услуги.

2. При нажатии в левом верхнем углу на иконку меню появится выбор из пункта «Руководство» и «О программе». 
При выборе одного из них откроется документ с соответствующей информацией о нём.
